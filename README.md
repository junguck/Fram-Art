# Fram'Art

DESCRIPTION

C'est une plateforme web permettant aux artistes et aux développeurs de collaborer afin de promouvoir des œuvres d’art auprès du grand public.



OBJECTIFS

-Mettre en relation artistes et développeurs
-Permettre la publication d’œuvres
-Offrir un espace de collaboration
-Faciliter la découverte artistique



FONCTIONALITES

-Authentification
-Publication d’œuvres
-Likes & commentaires
-Profil utilisateur
